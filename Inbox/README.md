# 📥 Inbox

> **快速捕捉区** — 随手记录，无需整理，定期清空到对应知识分类

---

## 使用说明

1. **捕捉**：任何新想法、链接、代码片段、读到的内容，直接丢到这个目录
2. **整理**（每周一次）：
   - 技术知识 → 移动到 `Knowledge/` 对应目录
   - 资源链接 → 更新到 `Sources/` 对应索引
   - 完整的技术地图 → 移动到 `Atlas/`
3. **保持清空**：Inbox 不是存档，整理完立即移走
