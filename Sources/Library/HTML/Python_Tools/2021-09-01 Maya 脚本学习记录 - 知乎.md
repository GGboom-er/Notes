█ **cmds 中的一些必要函数**

* **ls 命令:返回场景中对象的名称(也可以是类型名称)**

```
# List all selected objects
cmds.ls( selection=True )

# List all hilited objects
cmds.ls( hilite=True )

# List last selected object
cmds.ls( selection=True, tail=1 )

# List all objects named "sphere1". Note that since sphere1 is
# instanced, the command below lists only the first instance.
cmds.ls( 'sphere1' )
```

* **sets 命令：该命令用于创建一个集合，查询集合的某些状态，或执行更新集合成员的操作。 集合是对象、属性或对象组件的任意集合的逻辑分组。 集合是依赖节点。 从对象到集合的连接定义集合中的成员。详细内容可以参开如下链接**

[[Maya] 关于集(set)功能​modelerrigs.wordpress.com/2019/02/09/mayasetsfunction-cn/![](./2021-09-01 Maya 脚本学习记录 - 知乎_files/v2-3a835a98f248d57a98171139257c9e7e_ipico.jpg)](https://link.zhihu.com/?target=https%3A//modelerrigs.wordpress.com/2019/02/09/mayasetsfunction-cn/)

```
import maya.cmds as cmds

# create some objects
cmds.sphere( n="sphere1" )
cmds.cone( n="cone1" )

# create a set with whatever is currently active
cmds.select( 'sphere1' )
newSet1 = cmds.sets()
cmds.select( 'cone1' )
newSet2 = cmds.sets()
```

* **duplicate 命令 ：复制一个对象，如果没有给定特定对象，则复制被选中的对象。**

```
# Create a duplicate of the group
cmds.duplicate( 'group1' )
```

* **listRelatives 命令 ：列出DAG（Directed Acyclic Graph）对象的父节点和子节点们**

> 注意：返回的是字符串数组，就是父/子节点的名称

```
# list all parents of shape
# (The result of the command is shown)
cmds.listRelatives( shapes[0], allParents=True )
```

* **listConnections 命令：返回连接到给定对象的所有指定类型的属性/对象的列表**

```
import maya.cmds as cmds

cmds.sphere( ch=True, n='BALL' )
cmds.setKeyframe()
# List all connections to BALL
list = cmds.listConnections('BALL')
# List only incoming connections from BALL.tx
cmds.listConnections( 'BALL.tx', d=False, s=True )
# List connections from BALL to nodes of type 'transform'
cmds.listConnections( t='transform' )
# List connections on BALL, ignoring unit conversion nodes
cmds.listConnections( 'BALL', scn=True )
```

* **listHistory 命令：返回对象所经过的所有构造历史的节点**

```
import maya.cmds as cmds

cmds.curve( d=3, p=[(-3, 0, 0),(-1, 0, 6),(6, 0, 8),(8, 0, 2)], k=[0,0,0,1,1,1], n="snake" )
cmds.instance( n="rattler" )
cmds.revolve( 'rattler', ch=True, n="charmer" )
cmds.revolve( 'snake', ch=True, n="medusa" )

cmds.listHistory()
# Result:[u'medusaShape', u'revolve2', u'snake|curveShape1'] #
cmds.listHistory( 'charmer' )
# Result:[u'charmerShape', u'revolve1', u'rattler|curveShape1'] #
cmds.listHistory( 'medusa', lv=1 )
# Result:[u'medusaShape', u'revolve2'] #
```

* **listAttr 命令 : 该命令列出节点的属性。 如果没有指定标志，则列出所有属性。**

```
#获得blendshape的所有target
targets = cmds.listAttr(blendShape + ".w", m=True)
```

* **blendShape 命令：创建一个blendShape变形器**

```
cmds.blendShape( 'curve1', 'curve2', 'curve3' )
```

* **parentConstraint 命令：约束对象的位置和旋转，使其行为就像目标对象的子对象。**

```
# Rotate cube1 to the rotation of cone1
cmds.parentConstraint( 'cone1', 'cube1' )
```
