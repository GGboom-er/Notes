# Brave Rabbit 提供了一个快速指南，可根据 Maya 皮肤簇中的权重拆分混合形状。

勇敢的兔子（Ingo Clemens）分享了一些只有少数人知道但可以为装配工提供巨大帮助的东西。快速提示显示了如何使用权重服务器节点（一种可以从 Brave Rabbit 免费获得的工具）  通过皮肤簇权重将[单个混合变形目标拆分为多个区域。](https://lesterbanks.com/2019/01/rampweights-lets-you-split-deformer-weights-in-maya/)

“生成的目标形状是原始完整形状的简单副本，weightsServer 节点将基于影响的权重传输到这些目标。” 结果是实时连接，这意味着您可以在工作时编辑权重并查看混合变形结果。

删除 weightsServer 节点后，权重将保留在混合变形节点上。

## Maya 的权重服务器节点。

You can get the WeightServer node as part of the free download for Brave Rabbit’s RampWeights tool. Ramp weights node gives you fast access to splitting blend shape targets or weighted deformers along any axis. Both the nodes are also [part of the SHAPES plug-in](http://www.braverabbit.com/shapes/).
