## 人脸 (FACS)扫描库

面部扫描库是我们收藏中增长最快、最重要和最全面的库。包含 1150 多个个人的面部扫描图像，每个图像都带有 21 种 FACS（面部动作编码系统）表情。除了独特的肖像，我们还提供各种表情，涵盖不同种族、年龄、体型、身高和体重。

每个模型都在受控的标准化环境中进行精心的 3D 扫描，以确保数据的一致性和质量。最终生成了超过 24,000 个经过精确缩放、定位和颜色校准的扫描件。

该库为研究人类面部和表情、为电影或视频游戏创建逼真的角色提供了宝贵的资源。一致性和高制作标准使其成为人工智能或机器学习研究的理想数据集。

[了解更多](https://triplegangers.com/blog/technology/face-scan-library-facs)

## 全身扫描库

全身扫描库是最新且快速扩展的系列。目前有超过 210 个模特，每个模特都呈现 23 种 BACS（身体动作编码系统）姿势，旨在将人体形状分类为各种基本姿势，同时考虑到年龄、健康状况、性别和体型等因素。此外，我们还提供各种姿势，包括舞蹈、格斗、古典和时尚姿势、放松姿势、芭蕾舞等等。

我们的图书馆收藏了超过 12,500 张独特的全身扫描照片，展示了人体形态的多样之美。通过突破扫描技术的极限，结合我们独特的处理流程，我们大大减少了繁重的后期清洁工作，为您的创意项目提供了逼真的基础。

无论您从事动画、游戏、虚拟现实还是其他任何需要高保真人体模型的项目，我们的全身扫描库都能提供无与伦比的资源。

## 手扫描库

手部扫描库包含 230 个模型，代表了不同种族、生物和身体特征的组合。每个模型都捕捉了我们设计的 62 种姿势，以涵盖人类手部运动的全部范围。

为了创建这个图库，我们必须构建一个专门的扫描装置来捕捉人类手部的高细节。这个图库包含超过 14,000 个独特的扫描，每个都捕捉到了手部解剖和动作的复杂细节。

我们开发这个图书馆是为了创建一个独特的资源，以便研究人类手部的解剖结构和运动。手部是我们身体中经常被忽视但又至关重要的部分，对于交流和表达情感至关重要。
