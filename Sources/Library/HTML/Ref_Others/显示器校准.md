显示器校准

|  |  |
| --- | --- |
| [3dRender.com](http://www.3drender.com/index.html) | **校准您的显示器** |
|  |  |
| **1.** | 确保您的显示器没有任何屏幕眩光或反光。反射在屏幕上的灯或窗户会阻止您准确地看到任何图像，尤其是暗色调。光线昏暗的房间，显示器后面只有一点光，通常是 3D 灯光工作的最佳环境。 |
| **2.** | 确保您可以看到从深色到浅色的各种色调。对于初学者，您应该能够清楚地看到这张图片中亮色和暗色矩形之间的区别，并阅读深色和浅色文本中的数字 1 到 8： |
| **3.** | 检查显示器的伽马。从远处看，此图像的实心中心应与条纹边缘区域具有相同的亮度： |
| **4.** | 将桌面墙纸和应用程序背景颜色调整为中性灰色或黑色。显示器上鲜艳的颜色会让您以不同的方式看待您的工作。 |
| [后退](http://www.3drender.com/index.html) | 3dRender.com 的内容 版权所有 1995-2006 Jeremy Birn。 |
