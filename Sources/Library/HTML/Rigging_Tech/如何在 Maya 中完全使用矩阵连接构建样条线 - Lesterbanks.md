# Cole O'Brien 展示了如何在 Maya 中制作矩阵样条曲线。

Autodesk Maya 2020 包含新的 [矩阵驱动变换](https://lesterbanks.com/2019/12/a-look-at-mayas-new-offset-parent-matrix/) 和操作节点，可以清理许多绑定设置。这意味着更少的节点蜘蛛网、更干净的装备、更整洁的大纲以及更快的艺术家评估。在过去的一年中，Cole O'Brien 尝试完全根据 Maya 的矩阵连接构建样条线，并详细介绍了他的发现。 

在此处查看 [Maya 中的矩阵样条以获取文章](https://coleobrien.medium.com/)。

## 有关矩阵驱动转换的更多信息。
