Cable constructor 
This script is extremely useful for creating custom cables. It speeds up the 
workflow of making a spline and attaching meshes on there by a lot. All you 
need is some building blocks. 
 
The script features shortcuts to spline drawing tools. You can easily create and 
modify a cylindrical cable.  
On each side of the cable, you can attach a mesh. This by selecting a specific 
mesh from the scene, or by cycling through a collection of meshes, to find the 
perfect fit. 
The same goes for meshes along the spline. These have much more 
customization and are updated live. Cycling 
through meshes won’t affect parameters. 
 
Besides adding regular meshes from the scene, you 
can generate helix splines that warp around your 
cable. These spirals are customizable and easy to 
add/ remove.  
 
Finally, you can quickly convert your cable 
to high poly and low poly, to have a 
preview. 
 
 
There are help buttons to quickly explain what each segment does.
